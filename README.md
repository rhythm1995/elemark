# elemark

**This is a markdown editor by Electron and Vue.js.**

## To Use

To clone and this

```bash
# Clone this repository
git clone https://github.com/rhythm1995/elemark
# Go into the repository
cd elemark
# Install dependencies
npm install
# Run the app
npm start
```
